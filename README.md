## ğŸš€ Getting Started

### Quick Start
1. Open `index.html` in a modern web browser
2. Start designing!

### Using the AI Chat
1. Click on the **AI Chat** tab on the left
2. Type what you want to create (e.g., "Create a blue house with a red roof")
3. Press Send or hit Enter
4. Watch the AI build your design!

### Manual Design
1. Click on the **Manual** tab
2. Choose a shape from the dropdown
3. Click "Add Shape"
4. Select it in the Objects tab to edit properties
